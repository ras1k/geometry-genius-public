Answer To The Questions

DOM is a the short form of Document Object Model. The purpose of DOM is to provide a way for programs to dynamically access and modify the content and structure of a web page. Without DOM, web pages would be static and unresponsive. DOM allows web developers to create dynamic and interactive websites and applications which makes the web more engaging and useful for users.

The Dom Methods are

  • document.getElementById() - It selects id
  • document.getElementsByClassName() - It selects all element by class name
  • document.getElementsByTagName() - It selects all elements by the selected tag name
  • document.querySelector() - It selects the first element of CSS element
  • document.querySelectorAll() - It selects all itemss of CSS element

Event Bubble is a mechanism of DOM. It means when an event is triggered it goes straight its ancestors maintaining the hierarchy, and also triggers them. For example, if you have a button inside a div inside a section, and you click on the button, the click event will first trigger on the button, then on the div, then on the section, and finally on the document itself.

A callback function is a function that is passed as an argument to another function. A common use case for callback functions is in event handling. We will use callback function to write lesser codes by using the same function again and again. Using callback functions allows for more flexible and modular code.
